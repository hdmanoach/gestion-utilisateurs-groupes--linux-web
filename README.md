# Gestionnaire d'Utilisateurs Django

Le Gestionnaire d'Utilisateurs Django est une application web pour gérer les utilisateurs et les groupes sur un système Linux. Elle fournit une interface simple et intuitive pour les tâches courantes de gestion des utilisateurs et des groupes.

## Fonctionnalités

### Gestion des Utilisateurs
- Créer de nouveaux utilisateurs
- Supprimer des utilisateurs existants
- Lister tous les utilisateurs avec une fonction de recherche
- Changer le mot de passe d'un utilisateur (y compris une option "mot de passe oublié")

### Gestion des Groupes
- Créer de nouveaux groupes
- Supprimer des groupes existants
- Lister tous les groupes avec une fonction de recherche
- Ajouter un utilisateur à un groupe spécifique
- Ajouter un utilisateur au groupe `sudo`
- Retirer un utilisateur d'un groupe
- Afficher les groupes auxquels un utilisateur appartient
- Afficher les utilisateurs membres d'un groupe donné

### Surveillance et Logs
- Afficher les utilisateurs actuellement connectés au système
- Système sécurisé d'accès aux logs par email (demande de code, vérification, renvoi de code)
- Afficher le contenu des logs au format HTML
- Afficher l'historique des informations

### Interface Utilisateur
- Menu principal de navigation
- Page d'erreur 404 personnalisée
